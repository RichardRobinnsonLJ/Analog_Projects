# ⚡ CMOS Analog IC Design

This repository contains the design and simulation of fundamental **CMOS analog circuits** using **Cadence Virtuoso (90nm technology)**.

---

## 🚀 Circuits Included

* Common Source Amplifier
* Common Gate Amplifier
* Common Drain Amplifier
* Cascode Amplifier
* Differential Amplifier
* Current Mirrors

---

## 🧠 Key Concepts

* MOSFET Biasing
* Small Signal Analysis
* Gain & Bandwidth Trade-offs
* Differential Operation
* Current Mirroring

---

## 🛠 Tools Used

* Cadence Virtuoso
* GPDK 90nm CMOS

---

## 📸 Contents

Each project includes:

* Schematic
* Simulation results (Transient & AC)
* Layout
* Post-layout verification

---

## 🎯 Goal

To build a strong foundation in **Analog IC Design**, forming the basis for advanced circuits like **Op-Amps and ADCs**.
